# [WORK IN PROGRESS] sloth.js

A modern javascript library that empowers native Array's functionality with extra methods that everyone loves.

```javascript
import sloth from 'sloth';

const arr = sloth([]);

arr.group('rating');
arr.group('properties', 'neighborhood');

arr.toSet();
```
